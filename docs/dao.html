Power to the people.

You make Podium.

Podium DAO creates fair, future proof and independent system to give people power and shape the future of free social media. And as Elon said:

It's about the future of civilization, not making money.

Everyone is a shareholder

Every created profile gets 1 share. That's a base for making an impact which can be expanded later.

You help, you decide

By voting on new features or writing a code you earn shares.

  • Each merged pull request on Github awards 10 shares
  • Each vote on new feature awards 1 share

The more shares the more voting power.

No board and other 3rd parties

Podium operates on flat structure where no 3rd party is involved in decision making.

The Lifecycle

  1. New feature is added to waitlist
  2. Once it's got 100 upvotes from shareholders it gets into main pipeline
  3. Each feature in the pipeline is being voted on for 2 weeks by shareholders
  4. If voting is passed (>50% votes) feature gets implemented on all platforms
  5. Feature is being immediately released.
